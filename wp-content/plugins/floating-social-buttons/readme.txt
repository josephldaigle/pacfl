Floating Social Buttons.

== Description ==

Floating Social Buttons Provide an easy way to add floating social sharing button.

<strong>Floating Social Buttons plugin allow to select following button.</strong>
<p>1. Facebook Like.</p>
<p>2. Facebook Share.</p>
<p>3. Tweet.</p>
<p>4. Google Plus.</p>
<p>5. Linkedin.</p>
<p>6. Digg.</p>
<p>7. StumbleUpon.</p>
<p>8. Pinterest.</p>

<strong>Floating Social Buttons having following Setting options</strong>

1. You can choose which button to display.
2. You can select position of button.
3. You can hide this button on mobile device.

<strong>This is simple and perfect plugin. </strong>


== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

<strong>Do the social buttons of this plugin load asynchronously? </strong>
